# ping-pong-dash

## about
- VCに入室したらどこかのTCに報告が行くで。
- 一度入室した人がいるとembedを更新するようになってるで。全員抜けてもう一度入ると新たなembedが生成されるで。
## setup
1. requirements.txtの依存関係をインストールするんやで。
2. .env.sampleを参考に環境変数をセットするで。
2. ターミナルで`python3 main.py`
## Enviroment

|OS|対応状況|
|:--:|:--:|
|Windows|しらん|
|macOS|わからん|
|GNU/Linux|ArchLinux/amd64で動作済み|
